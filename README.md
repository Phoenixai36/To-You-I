## What is T.U.I?

**To:You&I** is a glitch-professional **terminal HUD** for orchestrating AI coding agents, workspaces and shells вҖ” all from a single, unified cockpit inside your terminal.

Built with **Go + Bubble Tea + Lip Gloss**, designed for developers who run multiple AI agents (Claude, Goose, Aider, OpenCodeвҖҰ) across multiple projects simultaneously and need a clean, fast, keyboard-driven control room.

## Features

- **Workspace Rail** вҖ” switch between project workspaces instantly (`вҶҗ вҶ’`)
- **Agent Sidebar** вҖ” see all agents and their live status at a glance
- **Activity Feed** вҖ” real-time timeline of agent events and decisions
- **Agent Detail Panel** вҖ” review and respond to agent prompts without leaving the terminal
- **Command Palette** вҖ” `Ctrl+P` to run tasks, open shells, ask agents
- **Shell Agnostic** вҖ” works with PowerShell, pwsh, zsh, bash, fish
- **WezTerm Native** вҖ” first-class support for WezTerm workspaces and panes
- **Dark Phoenix Glitch Aesthetic** вҖ” because your terminal should look as good as your code

## Keybindings

| Key | Action |
|-----|--------|
| `вҶҗ вҶ’` | Switch workspace |
| `вҶ‘ вҶ“` | Select agent |
| `Enter` | Open agent detail / focus workspace |
| `i` | Send input to agent |
| `l` | View agent logs |
| `r` | Restart agent task |
| `Ctrl+P` | Open command palette |
| `Ctrl+W` | Close current panel |
| `F1` | Help |
| `q` / `Esc` | Back / Quit |

## Quick Start

```bash
# Clone
git clone https://github.com/Phoenixai36/To-You-I.git
cd To-You-I

# Install dependencies
go mod tidy

# Run
make run

# Or directly
go run ./cmd/tuicortex/
```

---

## Configuration

Copy `.env.example` to `.env` and adjust:

```env
# Server
TUI_PORT=7331
TUI_HOST=localhost

# Default workspace
TUI_DEFAULT_WORKSPACE=main

# WezTerm integration
TUI_WEZTERM=true

# Theme
TUI_THEME=dark-phoenix
```
